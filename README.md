### General info
* WM: i3
* Bar: polybar
* Notification: dunst
* Menu: dmenu / rofi
* Window alternate: rofi
* Terminal: kitty
* Editor: neovim
* SH: zsh
* Background manager: feh
* ~Multiple language runtime versions manager: asdf~
* File manager: ranger
* Player: mplayer
* PDF View: mupdf
* ~Virtualization: qemu~
* ~Screenshot: gnome-screenshot~
* Lockscreen: i3lock
